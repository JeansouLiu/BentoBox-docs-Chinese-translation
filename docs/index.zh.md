# BentoBox

BentoBox 是一个岛屿生存类游戏的核心插件，它可以提供像 SkyBlock, AcidIsland, OneBlock 那样的游戏模式. BentoBox 是集岛屿地形生成, 团队管理, 岛屿保护等功能于一体的插件. 更多功能还可以通过添加 BentoBox 扩展来实现 (你可以理解为迷你插件). 服主们可以自由地对插件进行个性化配置.
